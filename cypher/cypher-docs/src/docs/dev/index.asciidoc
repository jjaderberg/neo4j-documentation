[abstract]
--
This is the Cypher manual for Neo4j version {neo4j-version}, authored by the Neo4j Team.
--

The main parts of the manual are:

* <<cypher-intro>> -- Introducing the Cypher query language.
* <<query-syntax>> -- Learn Cypher query syntax.
* <<query-clause>> -- Reference of Cypher query clauses.
* <<query-function>> -- Reference of Cypher query functions.
* <<cypher-schema>> -- Working with indexes and constraints in Cypher.
* <<query-tuning>> -- Learn to analyze queries and tune them for performance.
* <<execution-plans>> -- Cypher execution plans and operators.
* <<cypher-deprecations-additions-removals-compatibility>> -- An overview of language developments across versions.
* <<cypher-glossary>> -- A glossary of Cypher keywords, with links to other parts of the Cypher manual.


_Who should read this?_

This manual is written for the developer of a Neo4j client application.
